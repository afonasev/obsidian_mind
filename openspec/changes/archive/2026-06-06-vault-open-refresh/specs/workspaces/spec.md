## MODIFIED Requirements

### Requirement: Список пространств

Панель SHALL отображать пространства активного vault вертикальным списком; каждый элемент SHALL показывать имя пространства (= имя его папки). Активное пространство MUST быть визуально выделено. При отсутствии активного vault панель SHALL не показывать список пространств, а приложение SHALL показывать приглашение открыть vault.

#### Scenario: Отображение списка с выделением активного

- **WHEN** активный vault открыт и в нём есть пространства, одно из которых активно
- **THEN** в списке отображаются имена всех пространств vault, активное визуально отмечено

#### Scenario: Нет активного vault

- **WHEN** активного vault нет
- **THEN** список пространств не отображается, показано приглашение открыть директорию-vault

#### Scenario: Vault открыт, пространств нет

- **WHEN** активный vault открыт, но пространств в нём ещё нет
- **THEN** список пуст, под ним доступна кнопка создания, канвас показывает подсказку создать пространство
